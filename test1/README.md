## 流程图1：考试及成绩管理流程
``` flow1
@startuml
title 考试及成绩管理流程
|教务处|
start
:考试安排;
:考试安排表;
|教师|
:出卷;
if(A,B试卷)
|教师|
else
:打印审批表;
|系主任|
:审批签字;
:打印审批表;
endif
|教师|
:打印试卷;
:试卷;
|学生|
:参加考试;
:答题;
|教师|
:阅卷出成绩;
if(成绩单)
|教务处|
:有不及格的?;
->有;
if(安排补考)
:补考安排表;
detach
endif
else
|教师|
:答卷;
:装订存档;
endif
:期末流程结束;
stop
@enduml
```
