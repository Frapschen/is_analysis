## 2. 参与者说明：

###     2.1 采编员

主要职责是：为图书馆购买新的图书并对图书进行编码

###     2.2 图书管理员

主要职责是：管理图书馆所有的书籍借出与归还

###     2.3 读者
    
主要职责是：在图书管中借阅图书与归还图书

###     2.4 系统维护员
    
主要职责是：在图书馆中管理读者与维护图书管理系统
